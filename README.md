**Вопрос аналитика:**  
«Давайте начистоту. Серьезные авторы привыкли к традиционной валюте — признанию в Scopus. Но в старой системе признание автора — это просто запись в реестре, которая никак не защищает его от завтрашнего дня. В нашей архитектуре прохождение фильтра мгновенно превращает труд в осязаемый институциональный актив.
